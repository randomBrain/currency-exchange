# CurrencyExchange
This app is written on Angular 9. Used Angular material as UI library. It have 3 sections:
1. Calculator - it coconverts currencies form one to another based on latest available exchange rates;
2. Rates - rates can be viewed by date based on currency;
3. History - view historical data in chart by currency;

## Usage
1. Clone this repo;
2. Run `npm ci`
3. Run `npm run build:prod`
4. Run `start-dev-server`


This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.0.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
